# Product Frameworks Reference

A collection of detailed guides for the most valuable product management frameworks, synthesized from Lenny's Podcast episodes.

## How to Use This Folder

Each file provides a comprehensive deep-dive into a single framework. Use these when you need:
- **Full context** on how a framework works
- **Step-by-step instructions** for applying it
- **Examples** of the framework in action
- **Common pitfalls** to avoid

For quick reference, see [product-best-practices.md](../product-best-practices.md).
